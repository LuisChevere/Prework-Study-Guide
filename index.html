Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The Body element represents the visible content shown to the user.
  • Most basic building blocks of websites.

CSS

  • a margin indicates how much space we want around the outside of an element.
  • a padding indicates how much space we want around the content inside an element.
  • Language used to describe the presentation of a website created with HTML.

Git

  • A free open source version control system, allows developers to collaborate on projects without overwritting each others work.
  • Git status: check what branch we are currently on.
  • Git check out -b branch-name: creates a new branch and switches to it.

Javascript

  • Lightweight programming language used to provide logic and enteractivity for the websites.
